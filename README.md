# sh-config
My basic shell configs for different *nix systems.

## What it contains
This repo contains basic shell configurations and sample functions for Operating System used by me.
Although any given shell should behave the same on every OS, there are some differences in available
commands, functions, preinstaled software etc. Moreover there are differences between versions.

Content of this repo was build based on my knowledge and knowlnedge shared via internet. Therefore
Some files can have it's own license. Before usage / modification check if file has separate license.

Use it at your own risk.

## Configurations that influenced this repository

- Mac with BASH
- Mac with ZSH
- Debian Linux with BASH
- Debian Linux with ZSH
- Other popular configurations (not used frequently enough by me, to create separate dierectory)
- Some exotic configurations (having some fun)
